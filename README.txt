Étrend PWA

Feltöltés HTTPS tárhelyre:
1. Tömörítsd ki a ZIP-et, vagy töltsd fel a benne lévő fájlokat egy HTTPS-es tárhely gyökerébe.
2. iPhone-on Safariban nyisd meg az oldalt.
3. Megosztás → Főképernyőhöz adás.

A PWA telepítés után offline működik. A vízfogyasztás, napi jegyzetek és pipák az adott készülék localStorage tárhelyén maradnak meg.
